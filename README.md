# Time Loop Mechanics Simulation

## Project Description
This project simulates a time-travel mechanic allowing timeline branching and state reversion. It focuses on manual memory management and pointer manipulation using custom Linked Lists.

## Technical Details
* **Language:** Java
* **Core Logic:** Custom Doubly Linked List & Node classes.
* **Concept:** Timeline branching without standard collection libraries.

## How to Run
Compile and run the main class:
```bash
javac Main.java
java Main
